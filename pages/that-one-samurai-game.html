Game Description

That One Samurai Game is a game developed for the second Game Studio subject in the University of Technology Sydney. This was created in partnership with Ayelin Gist, Laith Noaman and Kyle Hammer. This game also had additional participants that we've callobrated towards the sound design of the gameplay and generation of soundtracks. This game involved you playing as a samurai to slash down as many enemies as you make your way to the boss.

The development of the game was an attempt in developing a high quality stylised game as our final studio project. However this project involved a litany of challengesd we've endeavoured during its development.

Game Features

That One Samurai Game included many different features and mechanics developed to add a spin to the sword fighting. As an attempt to develop a high quality game the features and mechanics of the game build are listed below.

  • Custom AI and dynamic targeting between player and enemy for non static combat during gameplay.
  • Multi-scene workflows allowing for optimised loading of scenes and reduced memory cost. Additionally enables loading of different seperated areas without loading screens.
  • Use of post processing effects to allow for stylised modification of scene colours.
  • Custom shaders for non standard effects for additional visual ques or effects for artistic style.
  • Rewind mechanic allowing for player to scrub back and forth in time from the point in death. This requires a further system of recording of essential values to allow for proper backtracking to a previous point in gameplay.
  • Custom and kitbashed 3D modelling for quick and cinematic level design.
